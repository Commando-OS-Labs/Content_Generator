Ad Generator

Create branded templates across content types, then generate variations

Commando360.ai Plugin Guide

Templates Tab — Social Ad Templates

Generate platform-ready ad layouts on the Figma canvas.

  • Content Types — Pick which ad formats to create (Single Posts, Carousel, Image Posts, Stories, etc.)
  • Theme — Dark or Light background. Select both to generate both variants.
  • Branding — Upload your logo mark and/or full logo. These get embedded into every template.
  • Background Pattern — Optional decorative pattern behind the content. Choose from geometric, premium, or illustration styles. Density controls how visible it is.
  • QR Code — Paste a URL to auto-generate a QR code on templates that support it.
  • Platforms — Check which social platforms you need sizes for (LinkedIn, Instagram, Facebook, X, YouTube). Each selected platform generates correctly sized frames.

Blog Tab — Article Layout Templates

Generate full blog article scaffolds at 1440px width.

  • 8 Templates — Hero Banner, Editorial, Case Study, Changelog, Comparison, Data Report, Immersive, and Tutorial. Each has a unique layout structure.
  • Theme & Pattern — Same as Templates tab. Dark/Light background with optional pattern.
  • Content fields — All optional. Leave empty to get placeholder text, or enter a Title and Author to pre-fill. Body sections (2–6) controls how many repeated content blocks the template generates.
  • Named layers — Every text block has a name like title, section_1_title, section_1_body so the Generate tab can bulk-replace content later.

Mockups Tab — Device Frames

Generate product screenshot mockups in realistic device frames.

  • Layouts — Phone Hero, Phone Trio (3 fanned), Cascade (2 overlapping), Phone + Laptop combo, Laptop Hero, Browser Window, or Responsive Set (browser + phone).
  • Device Color — Space Black (dark titanium) or Silver (natural titanium).
  • How to use — Create the mockup, then select the screen layer in each device and fill it with your screenshot. Copy the finished mockup into any template.

Generate Tab — Bulk Content Variations

Replace placeholder text across selected frames using data.

  • JSON / CSV input — Paste JSON directly or upload a CSV/JSON file. Each row becomes a variation.
  • How it works — Select one or more template frames on the canvas, then hit Generate. The plugin duplicates the frame for each data row and replaces text layers whose names match your data keys.
  • Column names — Use layer names as your CSV headers (e.g., headline, description, cta for ad templates, or title, section_1_title for blog templates).

Tips

  • Generate templates first (Templates or Blog tab), then use Generate tab to fill them with real content.
  • You can re-run Generate on the same frames with different data to create new variations.
  • All frames are created on the current Figma page with auto-spacing.
  • Custom SVG uploads work for background patterns — use the "Custom Upload" option.
